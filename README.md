# Bank Loan Analysis using Excel & SQL

##  Project Overview
This project analyzes bank loan data to understand loan distribution, risk categories, and EMI payment trends using SQL and Excel dashboards.

##  Tools Used
- Microsoft Excel (Pivot Tables, Dashboards)
- SQL (MySQL)
- GitHub

##  Key Analysis
- Loan amount analysis by loan type
- Risk categorization using loan-to-income ratio
- EMI payment status analysis
- Interactive Excel dashboard with charts and slicers

##  Project Structure
- data/: Raw CSV files
- sql/: SQL scripts for table creation and data insertion
- excel/: Excel dashboard file
